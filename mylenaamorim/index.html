Git e Github

Git é um sistema de controle de versão de arquivos. Através deles podemos desenvolver projetos na qual diversas pessoas
podem contribuir simultaneamente no mesmo, editando e criando novos arquivos e permitindo que os mesmos possam existir sem
o risco de suas alterações serem sobrescritas. Os principais comandos do Git são: Git Clone que é utilizado para baixar o código-fonte existente de um repositório remoto (como o Github, por exemplo) -> git clone Git Branch que pode ser utilizado para criar, listar e excluir branches: Para criar: -> git branch Para ver os branches: -> git branch or git branch --list Para deletar: -> git branch -d Git Checkout que é usado principalmente para alternar de um branch para outro. Também podemos usá-lo para verificar arquivos e commits: -> git checkout Git Status que fornece todas as informações necessárias sobre o branch atual: -> git status Git Add que serve para incluir as alterações de um arquivo em nosso próximo commit. Para adicionar um arquivo: -> git add Para adicionar tudo de uma vez: ->git add -A Git Commit Este comando é como definir um ponto de verificação no processo de desenvolvimento, para o qual você pode voltar
mais tarde, se necessário: -> git commit -m "mensagem explicando a mudança no código" Git Push envia e salva suas confirmações no repositório remoto: -> git push Para fazer upload do branch com o seguinte comando: -> git push -u origin O Github é um serviço web que oferece diversas funcionalidades extras aplicadas ao git. Resumindo, você poderá usar gratuitamente
o github para hospedar seus projetos pessoais. Além disso, quase todos os projetos/frameworks/bibliotecas sobre desenvolvimento
open source estão no github, e você pode acompanhá-los através de novas versões, contribuir informando bugs ou até mesmo enviando
código e correções. Para criar um repositório: -> git init Para saber o status do repositório: -> git status Para adicionar todos arquivos e pastar ao repositório: -> git add . Para commitar: -> git commit -m "" Para clonar: -> git clone
