Git e Github

O git é uma ferramenta que auxilia no versionamento de um ou vários projetos e utiliza de snapshots para gravar os status desses projetos, quando não há alteração de um arquivo de uma versão anterior para a próxima, o git utiliza de um link que remete a versão atual do arquivo. Já o github é o local na web onde se guarda os projetos versionados pelo git tornando assim, o trabalho remoto auxiliando os usuários a poupar o recurso de memória do seu dispositivo, sendo possível também, receber ajuda e melhorias do seu projeto de pessoas do mundo inteiro, contribuir com o trabalho de outras pessoas e configurar seu projeto como privado. Para configurar o git com o seu nome, e-mail e o editor de sua escolha, são utilizados os seguintes comandos: “git config –global user.name “seu usuário”” “git config –global user.email “seu e-mail”” e “git config –global “core.editor”“, após as configurações já é possível utilizar os recursos do sistema git. Para criar uma pasta, é utilizado o comando: “mkdir “nome da pasta”“ Para remover um arquivo: “rm “nome do arquivo”” Listar os arquivos da pasta em que se encontra: “ls” Para inicializar o git o comando utilizado é: “git init” Para clonar um repositório é usado: “git clone “url do projeto”” Para a criação de uma nova branch com seu nome: “git checkout –b “seu nome”” Para renomear uma pasta ou arquivo: “git mv “nome atual do arquivo” “novo nome do arquivo”” Para salvar as alterações do seu arquivo: “git add --all” ou “git add .” e “git commit –m “comentário”” Para enviar sua branch para a nuvem: “git push origin “nome da branch”” Hélissan Christiansen Silva
